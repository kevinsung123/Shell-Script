#### 쉘변수 
- $# : 명령입력 라인에서 입력된 인수의 개수 
- $* : 명령입력 라인에서 입력된 인수 전체내용
- $$ : 프로세스id
- $! : 실행시킨 백그라운드 프로세스 id
- $? : 실행한 프로그램이 종료되면서 리턴한 값을 할당받아 가지고 있다 
	-  0 : 제대로 종료
	-  그외 : 비정상종료  

### read 
- 실행될 떄 사용자로부터 입력 요청
```
#!/bin/sh echo "grep에 사용될 옵션 입력, -l 또는 -s를 입력해 주세요." 
read option1 echo "검색하고자 하는 문장을 입력해 주세요. \c" 
read option2 
grep $option1 $option2 *  
  
```
